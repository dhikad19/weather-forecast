## Weather App Forecast

Simple weather forecast, using an API and React.

## This App Using

1. HTML
2. CSS
3. API
4. React
5. JavaScript

## Run the Program

Type in Terminal
```sh
   npm install
   ```
```sh
   npm start
   ```
